# FinTech-Fraud-Analysis
Overview

This project analyses financial transaction data to uncover fraud patterns, identify high-risk activity, and generate insights that can improve fraud detection strategies.
The dataset contains anonymised credit card transactions, labelled as fraudulent or legitimate.

This project demonstrates skills in:

Data cleansing

Exploratory analytics

SQL querying

Dashboard development in Power BI

Insight communication
